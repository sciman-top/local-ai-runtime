## Wave 1 smoke fixtures

These fixtures keep the `W1-T06` sample task set repo-owned and deterministic.

- `manifest.json`: sample registry and category mapping.
- `tasks/`: one task per Wave 1 category.
- `responses/`: scripted fake-worker output captured as artifact content.

The fixtures are for repo-side fake-first smoke only. They do not prove live `Codex SDK` acceptance.
